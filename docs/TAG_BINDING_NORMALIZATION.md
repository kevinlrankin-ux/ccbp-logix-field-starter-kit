\# Tag Binding Normalization (Spec ↔ Tests ↔ Ladder)



\## Purpose

Allow the spec to define tags hierarchically while tests may reference flattened names.



\## Rule

Flattened format:

\- Spec: S03.xJamOK

\- Test:  S03\_xJamOK

\- Ladder tag alias: S03\_xJamOK (or a mapping layer)



\## Canonical Mapping Examples

\- S01.xCmdRun     ↔ S01\_xCmdRun

\- S03.xJamOK      ↔ S03\_xJamOK

\- S05.xFbRunning  ↔ S05\_xFbRunning
